Help 4U2

How do I customize this template?

Just click on the big "Customize this template now!" image up top, and a ProtonDrive folder should lead you to the zip file, with all the resources you need.

Do I need to give you any credit?

Nope!

Can I customize the graphics you used here?

Absofruitly! There are .psd files included in the 4U2 zip, which you can edit as you please.

About the template

This template is based on a grid display with some pre-styled sections. I'm not so good at making templates yet but I hope someone makes use of and enjoys the 4U2 template! :)

This also has a dark+light mode, which I tried to make easy to edit with the variable color palette.

Today on ICQ

Custom colors!

The color palette's main accent (right now, it's green) is all variable-based, so by changing the base accent, all the other colors match it.

About the webmaster

This template is all about you!

I modeled this template to look like ICQ from 2005. This template is to do with as you please: share it, fix it, give it a do-over, learn from it... anything you want! (40 word limit)

What 4U2 means

ICQ, when said out loud, means "I seek you," since it was a social website for people to meet each other. I've named this template 4U2, since not only is it for me... it's For You, Too! :)

Need help?

I hope the template's mostly intuitive to use but if not, send me a message. I would love to help out or even just make new friends with the indie web community.
